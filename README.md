# CENTRA - Infraestructura

Este repositorio centraliza la gestión de issues, documentación y configuraciones 
relacionadas a la infraestructura del proyecto CENTRA.

## ¿Qué se gestiona aquí?
- Configuración y mantenimiento del VPS
- Dominios y certificados SSL
- Variables de entorno y secretos (referencias, no valores)
- Scripts de despliegue y automatización
- Decisiones técnicas de arquitectura (ADRs)

## Repositorios del proyecto
| Repositorio | Descripción |
|---|---|
| `centra-backend` | API y lógica de negocio |
| `centra-infra` | Infraestructura y DevOps (este repo) |
