# CENTRA - Infraestructura y Gestión 🏗️

Este repositorio centraliza la administración, documentación técnica y configuraciones de infraestructura para el ecosistema de aplicaciones de **CENTRA**.

## 📌 Propósito
El objetivo de este espacio es gestionar todo lo que es transversal al código fuente, como:
- Configuración y mantenimiento de servidores (VPS).
- Gestión de dominios, DNS y certificados SSL.
- Estrategias de despliegue (CI/CD) y Dockerización.
- Documentación de arquitectura y decisiones técnicas (ADRs).

## 📂 Mapa del Proyecto
Actualmente, el ecosistema de **CENTRA** se divide en los siguientes repositorios:

| Repositorio | Estado | Descripción |
| :--- | :--- | :--- |
| `centra-infra` | 🟢 Activo | Gestión de infraestructura, DevOps y este README global. |
| `centra-backend` | 🟡 En desarrollo | API Core, lógica de negocio y base de datos. |
| `centra-frontend` | ⚪ Próximamente | Interfaz de usuario (Web/Mobile). |
